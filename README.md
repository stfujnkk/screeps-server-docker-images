# screeps-server-docker-images
Screeps World server 's docker image

## Download

```bash
https://github.com/stfujnkk/screeps-server-docker-images/releases
```

## Build

```bash
./build
```

## Server launch

```bash
docker exec -it screeps-server bash
cd /screeps-server
npx screeps init
npx screeps start
```
